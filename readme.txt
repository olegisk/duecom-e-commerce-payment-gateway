== Description ==

Be sure to checkout our <a href="https://due.com/blog/request-access-use-due-payment-gateway-woocommerce/">tutorial</a> on using this plugin.

This plugin adds the Due.com E-Commerce Payment Gateway to WooCommerce for accepting **Credit Cards Payments** directly on checkout page.
This plugin uses Due API **v1** to charge credit cards. For better visualization of how it looks & works check screenshots tab.

= Features =
1. Simple Clean Code to accept Credit cards via Due Payments in WooCommerce
2. No technical skills needed.
3. Easy customization.
4. Single checkbox to set plugin in Sandbox/Live mode.
5. This plugin does not store **Credit Card Details**
6. This plugin requires SSL.
7. This plugin supports refunds **(Only in Cents)** in woocommerce interface
8. Contribute to future releases by sending us <a href="mailto:chalmers@due.com?subject=Feature%20Request">Feature Requests</a>!

== Installation ==

1. Upload 'duecom-e-commerce-payment-gateway' folder to the '/wp-content/plugins/' directory
2. Activate 'Due.com E-Commerce Payment Gateway' from wp plugin lists in admin area
3. Plugin will appear in settings of WooCommerce
4. You can set the addon settings from 
   wocommmerce -> settings -> Checkout -> Due.com Payments
5. **Make sure you have 2 Decimal Places in Pricing Options (WooCommerce > Settings > General > Currency Options) with Trailing Zeros Checked else the charge created by card will be incorrect**
